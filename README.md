### search-form

 клонировать проект: ```git clone https://github.com/EfymichevaKseniya/search-form.git```

## Запуск сервера
     cd server
     npm run start

     сервер запустится на http://localhost:8000/

## Запуск клиента
     cd client
     npm run start

     клиент запустится на http://localhost:3000/
